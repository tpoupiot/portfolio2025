## 1. présentation du projet

Le but de ce projet était de concevoir et de réaliser un site web de gestion de location de logements de vacances, inspiré du site Airbnb.

### étapes du projet :

- Effectuer un recueil des besoins
- Créer des user stories
- Réalisaer le maquettage du site
- Développer une base de données
- Réaliser le site web

## 2. méthode de travail

Dans ce projet, nous avons adopté une approche agile pour garantir une livraison rapide et efficace des fonctionnalités. Voici comment nous avons structuré notre méthode de travail :

### Méthodologie Agile

Nous avons utilisé la méthodologie agile pour nous adapter aux changements et améliorer continuellement notre produit. Cela nous a permis de rester flexibles et de répondre rapidement aux besoins des utilisateurs.

### Sprints

Le projet a été divisé en plusieurs sprints, chacun durant deux semaines. Chaque sprint comprenait les étapes suivantes :

1. **Planification du Sprint** : 
   - Définir les objectifs du sprint.
   - Sélectionner les user stories à réaliser.
   - Estimer le temps nécessaire pour chaque tâche.

2. **Développement** :
   - Les développeurs ont travaillé sur les tâches assignées.
   - Des réunions quotidiennes (stand-ups) ont été tenues pour suivre l'avancement et résoudre les obstacles.

3. **Revue du Sprint** :
   - Présentation des fonctionnalités développées aux parties prenantes.
   - Recueillir les retours et ajuster les priorités si nécessaire.

4. **Rétrospective du Sprint** :
   - Discuter de ce qui a bien fonctionné et des points à améliorer.
   - Mettre en place des actions pour améliorer le prochain sprint.

Cette approche nous a permis de livrer un produit de haute qualité tout en restant alignés avec les attentes des utilisateurs et des parties prenantes.
